== Welcome to kon10trat

kon10trat is a web appplication that allows you to configure a rails application to be then generated
and downloaded preconfigured with all the needed plugins etc.

== Getting Started

1. Register and Login to www.kon10trat.de.
2. Choose create new rails application.
3. Add plugins, gems, and git options.
4. Click generate application and download the generated app.
